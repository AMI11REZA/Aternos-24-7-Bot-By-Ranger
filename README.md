

## Installation
 first go to aternos and install Via Version Or ViaBackwards Plugin Depending On You'r Aternos Server Version (*If Its Higher Than 1.21.1 then install Via Backwards Only*) and after that go to options and enable cracked, and then click on the green code icon then select Download ZIP. then extraxt the zip file and go to index.js, enter your server ip and port. after that create a new responsibility then import the files you just extracted. then go to aternos and go to players tab, select AFKBot_ByRanger and set the game mode to spectator(you have to turn off forced gamemode on options for this to work!).
